GALERÍA DE ARTE MEXICANO - PAQUETE LISTO PARA NETLIFY

ARCHIVOS PRINCIPALES
- index.html = Inicio
- artists.html = Artistas
- shop.html = Tienda / Obras
- mariana-silva.html = Perfil de artista
- script.js = Hace clicables algunos botones

CÓMO SUBIR A NETLIFY
1. Entra a https://app.netlify.com/drop
2. Arrastra la carpeta completa o comprímela en ZIP.
3. Netlify generará una URL pública.

QUÉ EDITAR FÁCILMENTE
1. Textos:
   - Abre cualquiera de los archivos .html
   - Busca el texto que quieras cambiar
   - Sustitúyelo y guarda

2. Imágenes:
   - Este export usa muchas imágenes por URL externa.
   - Busca src="https://..." dentro del HTML.
   - Sustituye esa URL por otra imagen pública.
   - También puedes crear una carpeta images y usar rutas como:
     src="images/mi-imagen.jpg"

3. Correos de contacto:
   - Busca hola@tudominio.com y reemplázalo por tu correo real.

ENLACES YA CONECTADOS
- Menú principal entre Home / Artists / Shop
- Algunos botones clave ya navegan entre páginas
- Botones de consulta abren correo

RECOMENDACIÓN
Si después quieres dominio propio, lo conectas desde Netlify sin rehacer el sitio.
